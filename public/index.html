About EduChain

EduChain is a revolutionary blockchain-based assignment management system that ensures academic integrity through cryptographic verification. Our platform provides a secure, transparent, and tamper-proof environment for educational institutions.

Key Features:

  • Blockchain-secured file storage with SHA-256 hashing
  • MetaMask integration for secure transactions
  • Role-based access control system
  • Comprehensive audit trail
  • Excel export functionality for reports

How It Works:

Students upload assignments through our secure interface. Each file is hashed using SHA-256 and stored on the blockchain, creating an immutable record. Lecturers can verify submissions by comparing blockchain hashes and ensuring authenticity.
